# Анализ цен на жильё в Казахстане (апрель 2026)

**Проект для портфолио дата-аналитика**

В этом проекте я провела полный цикл анализа рынка жилья в Казахстане: от сбора и очистки данных до создания интерактивных дашбордов в Tableau и публикации портфолио.

##  Задача проекта
Сравнить стоимость нового жилья, перепродажи и аренды по городам Казахстана, выявить, где выгоднее покупать, а где — сдавать, и визуализировать результаты для бизнес-выводов.

##  Источник данных
Официальные данные с портала **statgov.kz** за апрель 2026 года  
*(самостоятельный сбор и первичная обработка)*

##  Использованные инструменты
- **Excel** — первичная обработка, формулы (разница между новостройкой и перепродажей, определение «Что дороже?», поиск MAX/MIN)
- **Tableau Public** — визуализация и создание дашбордов
- **GitHub** — портфолио проектов

##  Что я сделала
| Этап | Действие |
|------|----------|
| 1. Сбор данных | Скачала таблицу с ценами по всем городам РК |
| 2. Расчёты в Excel | Добавила колонки: разница между новостройкой и перепродажей, что дороже, MAX/MIN цены |
| 3. Визуализация в Tableau | Построила 4 графика + карту |
| 4. Выводы | Определила города с самой высокой/низкой ценой, разрыв между сегментами |
| 5. Публикация | Опубликовала дашборд на Tableau Public + код/описание на GitHub |

##  Ключевые визуализации (Tableau)

1. **Цены на перепродажу по городам** — горизонтальный бар-чарт с сортировкой
2. **Цены на новостройки по городам** — сравнение с перепродажей
3. **Аренда vs цена продажи (новостройка)** — scatter / dual-axis
4. **Карта Казахстана** — географическое распределение цен
5. **Сравнительная таблица** — город / тип сделки / разница

> Все графики интерактивны, с фильтрацией по городам и метрикам.

##  Основные выводы
- **Самая высокая цена нового жилья** — Алматы (733 375 ₸)
- **Самая низкая цена нового жилья** — Уральск (325 231 ₸)
- В большинстве городов перепродажа дороже новостройки (например, Шымкент, Актау)
- В некоторых городах (Актобе, Кокшетау, Костанай, Уральск, Петропавловск, Семей) новостройки дороже перепродажи
- Аренда максимальна в Алматы (5 824 ₸) и Астане (5 529 ₸), минимальна в Таразе (2 525 ₸)
